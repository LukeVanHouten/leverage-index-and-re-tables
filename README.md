# leverage-index-and-re-tables

### Objectives

- RE tables for every year, stadium, team (done)
- RE24 tables for every inning, and cumulative scores for player seasons (done)
- Linear Weights (done)
- Win Expectancy deltas for every possible situation (done)
- Leverage Index for every plate appearance in the Statcast era (done)
- Different types of LI for players
- Predict WE with LI etc.

### Notes

I am using Statcast data between 2015-2019 and 2021-2025. However, I am missing some of the newer features added to Statcast in the last year or so. This project was thus completed without them.
